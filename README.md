## 資料庫測驗
題目一與題目二都在資料庫測驗/資料庫測驗.txt

## API實作測驗
```
git clone https://github.com/derailment/asiayo-backend-pre-test
cd asiayo-backend-pre-test/API實作測驗
pip install virtualenv
virtualenv flask
flask/bin/pip install flask
chmod +x app.py
chmod +x test.sh
./app.py
```
在另一個terminal
```
./test.sh
```
